# Atomic Acres Pass 25A Defect Ledger — 2026-07-16

Pass 25A preserves the approved Pass 24 game feel while turning regressions and production failure paths into executable evidence.

## Classification rules

- **Confirmed:** reproduced from source or browser evidence.
- **Fixed:** correction is implemented and has a named regression check.
- **Harness:** verifier defect only; no gameplay defect inferred.
- **Deferred:** evidence exists, but correction would intentionally change approved presentation or feel.
- Technical green does not override Dave's owner A/B feel approval.
